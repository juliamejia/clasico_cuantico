# clasico_cuantico
este proyecto implementa alguno de los experimentos de la cuantica , como lo es el caso de las canicas: halla la posicion de cada canica despues de tantos clicks 
, experimento de la doble rendija:el usuario ingresa las probabilidades de que las balas se muevan de cada rendija, el vector inicial y el numero de disparos 
y probabilidades: las probabilidades que tiene un vector de estado
